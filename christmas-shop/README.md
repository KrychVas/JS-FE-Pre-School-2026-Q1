# 🎄 Christmas Shop

## 📝 Project Description
The Christmas holidays are just around the corner! It's time to buy gifts... but where?  
**"Christmas Shop"** is a multi-page web application consisting of two interactive pages. The project focuses on building a fully responsive, semantic, and pixel-perfect website based on a Figma design layout, enhanced with custom JavaScript functionality to create a cozy and seamless user experience for holiday shopping.

---

## 🛠 Key Skills
* Valid, semantic, and responsive web design;
* Clean, maintainable, and readable code;
* Exporting styles and assets from Figma;
* Using native JavaScript to implement interactive functionality.

---

## 📅 Project Development Stages
The task is divided into three stages, with a duration of 1 week for each:

1. **Part 1: Fixed Layout**
   * Creating the "Home" and "Gifts" pages based on the layout template.
   * Ensuring correct display for window widths of at least `1440px`.
   * Checking design consistency and overall layout accuracy.
2. **Part 2: Responsive Design**
   * Adding responsiveness for all devices down to a width of `380px` inclusive.
   * Ensuring the layout adjusts smoothly without any horizontal scrollbars (overflow issues).
3. **Part 3: Adding Functionality**
   * Utilizing JavaScript to bring the pages to life and implement interactive elements.
